# BoredApp
 
Это приложение - pet-project для подбора вида занятия, если человеку стало скучно. 

### BoredApp:

- Написано на UIKit
- Использует открытый сторонний Bored Api (https://www.boredapi.com/about)
- Использует паттерн MVC
- Xcode Version 14.3.1 
- IOS 16.4
